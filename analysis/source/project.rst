Project description:
====================
This project contains code for the post-processing of ROOT output of my AliPhysics RAA task. The framework is written in PYTHON using then PyROOT bindings, and contains code for reading the files, making projections of the THnSparses, applying corrections, plotting results, and many more. The package depends on the following PYTHON packages
  - PyROOT
  - SciPy
